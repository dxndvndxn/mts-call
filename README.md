# Как собрать проект

### `docker-compose up --build -d`

## Открыть в браузере чат Оператора

### `http://localhost:8080/`

## Открыть в другой вкладке чат Клиента

### `http://localhost:8080/client`

В открывшимся чате задать любой вопрос
